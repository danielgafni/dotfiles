# Dotfiles

This repository contains my personal configuration files for various Linux apps & tools:

 - zsh
 - neovim
 - alacritty
 - zellij
 - starship

All the colors & theming is set to Nord Theme.

# Installation

Run 

```shell
sh -c "$(curl -fsLS https://chezmoi.io/get)" -- init --apply danielgafni
```
or
```shell
./install.sh
```
